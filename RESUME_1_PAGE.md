## ? DÉMARRAGE RAPIDE

### Option 1 : GitHub Desktop

```
1. Installez GitHub Desktop
   ? https://desktop.github.com/

2. Créez un compte GitHub (gratuit)
   ? https://github.com/signup

3. Dans GitHub Desktop :
   - Publish repository (bouton)
   
4. Sur GitHub.com :
   - Créez une release (cliquez)

? C'EST AUTOMATIQUE !
```

---

### Option 2 : Google Drive

```
1. Créez un dossier dans Google Drive

2. Uploadez 3 fichiers :
   - version.txt (contenu: 1.0.0.0)
   - logiciel_impression_3d.exe
   - installateur (optionnel)

3. Partagez publiquement (clic droit)

4. Copiez les URLs dans UpdateManager.cs

? TERMINÉ !
```
